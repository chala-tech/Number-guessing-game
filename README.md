Number Guessing Game
A Python-based logic game with multiple difficulty levels.
Description
This is a simple, interactive command-line tool where the player attempts to guess a randomly generated number within a specific range. I developed this project to practice Python fundamentals, including loops, conditional logic, and user input handling.

 Features
Multiple Difficulty Levels: Choose between Easy, Medium, and Hard modes to change the range of numbers.

Input Validation: The game ensures the user enters a valid number.

Score Tracking: Tracks how many attempts it takes to find the correct answer.

Built With
Python: The core programming language used for logic and execution.

 How to Run
Ensure you have Python installed on your machine.

Download the number guessing game.py file from this repository.

and then run it in you selected  code editor.
